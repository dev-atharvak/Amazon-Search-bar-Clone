# Amazon Search Bar Clone

A simple frontend clone of the Amazon search bar built using HTML and CSS. This project was created to practice responsive layouts, styling, form elements, and UI design by recreating a commonly used component from Amazon's website.

---

## Features

- Amazon-inspired search bar design
- Search input field
- Category dropdown menu
- Search button with icon
- Clean and responsive layout
- Built using pure HTML and CSS

---

## Technologies Used

- HTML5
- CSS3

---
  
## Purpose

This project was developed for frontend development practice and to improve understanding of:

- Form elements
- Flexbox layout
- CSS styling
- UI cloning techniques
- Responsive web design
